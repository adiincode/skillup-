# 🚀 AI Skill Recommendation System

> An AI-powered Career Recommendation Platform that analyzes a user's existing skills, identifies skill gaps, recommends career paths, and suggests personalized learning resources.

--

---

#  Overview

Choosing the right career path is one of the biggest challenges for students and professionals.

This project uses Artificial Intelligence techniques to analyze a user's current skill set and provides:

- 🎯 Career Recommendations
- 📈 Career Match Score
- 📚 Personalized Learning Resources
- 🧠 Skill Gap Analysis

The application is developed using **Python**, **Pandas**, **Scikit-Learn**, and **Streamlit**.
